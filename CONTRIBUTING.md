## Prerequisites

- NodeJS 18+

## Setting Up Locally

- Create a `.env.local` file

```properties
# .env.local
DISCORD_CLIENT_ID=<discord-client-id>
DISCORD_TOKEN=<discord-token>
```

- Install the Node Modules

```sh
$ npm ci
```

## Start the Bot~

```sh
$ npm start
```

## Run the Tests

```sh
$ npm test
```
